Types of Plugins

The framework allows developers to create two types of plugins for XSplit Broadcaster: source plugins and extension plugins. Sources are added to the stage to be recorded or streamed live, while extensions occupy their own window and are given elevated privileges to manipulate sources and certain application properties.

Sources

A source is a plugin placed on the stage. Right-clicking the source opens its properties window. Using HTML/CSS/JavaScript, developers can create any kind of source, and a properties window for that source. For more information on developing sources, click here.

Extensions

Extensions are plugins on the global level. They are still developed using HTML, and have all the capabilities of sources and more. Please check out the API for the current set of implemented features, and drop us a line here to discuss any feature requests. A list of additional notes regarding extensions can be found here.
