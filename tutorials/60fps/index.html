60 FPS Browser Sources

By default, HTML sources run at a maximum of 30 frames per second. For more advanced uses, your HTML page may request to use up to 60 FPS instead.

This can be achieved by adding window.external.SetLocalProperty("prop:Browser60fps","1"); on your source.

You can do any of the following to enable 60fps on your HTML Sources:

Modify original HTML/JS

Add the custom script above anywhere on your script tag on your HTML or add it inside your JS file.

Inject custom JS

You can inject the code above through the Custom Code option on an HTML Source source property window.

After adding the custom script, this would automatically take effect and your source would refresh itself if needed.

Manually change Presentation file

You can also save the presentation where you have your HTML Source, and manually change the 60fps file property of your source.

First, save the presentation that contains your HTML Source, that would create a file with a BPres extension. Open this file on any text editor and search for Browser60fps property and set it to 1, save, then load the presentation again.

⚠Please be informed that this is currently an experimental feature, and may not work perfectly for different machines.
