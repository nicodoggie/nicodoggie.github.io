Screen class

This class serves to allow developers to add new screen regions or window regions to the stage in XSplit Broadcaster.

Members

addToScene 

param: (value?: number | Scene)

return: Promise<boolean>

Initializes the screen region selector crosshair so user may select a desktop region or a window to add to the stage in the current scene. Accepts an optional parameter value, which, when supplied, points to the scene where item will be added instead.
