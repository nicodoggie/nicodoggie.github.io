FlashItem class

The FlashItem class represents a flash item, which is any SWF file loaded to XSplit Broadcaster.

Inherits from: Core/Item

Implements: Core/IItemChroma, Core/IItemColor, Core/IItemLayout, Core/IItemTransition, Core/IItemAudio

All methods marked as Chainable resolve with the original FlashItem instance. Also, any audio setting, i.e. volume, mute, stream only may not be properly reflected in the item unless native flash audio support is enabled. (Tools menu > General Settings > Advanced tab)

Members

getCustomResolution 

return: Promise<Rectangle>

Gets the custom resolution (in pixels) for the item, if set, regardless of its layout on the mixer. Returns a (0, 0) Rectangle if no custom resolution has been set.

See also: Util/Rectangle

Check if right click events are sent to the item or not.

setAllowRightClick 

param: (value:boolean)

return: Promise<Item>

Allow or disallow right click events to be sent to the item. Note that you can only catch right click events using mouseup/mousedown

Chainable

Usage

item.setAllowRightClick(true).then(function(item) {
  // Promise resolves with the same Item instance
});
