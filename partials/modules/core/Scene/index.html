static initializeScenes 

return: Promise<boolean>

Load scenes that are not yet initialized in XSplit Broadcaster.

Note: For memory saving purposes, this is not called automatically. If your extension wants to manipulate multiple scenes, it is imperative that you call this function. This function is only available to extensions.

Usage

Scene.initializeScenes().then(function(val) {
  if (val === true) {
    // Now you know that all scenes are loaded :)
  }
})
