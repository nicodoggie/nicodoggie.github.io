HTMLItem class

The HTMLItem class represents a web page source. This covers both source plugins and non-plugin URLs.

Inherits from: Core/Item

All methods marked as Chainable resolve with the original HTMLItem instance. Also, any audio setting, i.e. volume, mute, stream only may not be properly reflected in the source unless native browser audio support is enabled. (Tools menu > General Settings > Advanced tab)
