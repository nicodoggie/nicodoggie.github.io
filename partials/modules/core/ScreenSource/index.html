ScreenSource class

The ScreenSource class represents the sources of the screen device items that has been added to the stage. A single source could have multiple items linke.... into it and any changes to the source would affect all items linked to it.

Each item is represented by the ScreenItem class. See: Core/ScreenItem

Inherits from: Core/Source

Basic Usage

var xjs = require('xjs');

xjs.Scene.getActiveScene().then(function(scene) {
  scene.getSources().then(function(sources) {
  for (var i in sources) {
      if (sources[i] instanceof XJS.ScreenSource) {
        // Manipulate your screen source here
        sources[i].setSilenceDetectionEnabled(true);
      }
    }
  })
})

All methods marked as Chainable resolve with the original ScreenSource instance.
