Transition class

The Transition class represents a preset transition within XSplit Broadcaster. This may be used to set the application's transition scheme when switching scenes, or to set an individual source's transition when its visibility changes.

Simply use one of the available Transition objects such as Transition.FAN or Transition.COLLAPSE as the parameter to the setTransition() method of an App instance, or a valid Item instance that supports transitions (this includes Core/CameraItem, Core/GameItem, and Core/HTMLItem.)
