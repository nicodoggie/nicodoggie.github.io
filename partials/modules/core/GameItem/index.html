GameItem class

The GameItem Class provides methods specifically used for game items and also methods that is shared between Item Classes. The Scene class' getItems method would automatically return a GameItem object if there's a game item on the specified scene.

Basic Usage

var XJS = require('xjs');

XJS.Scene.getActiveScene().then(function(scene) {
  scene.getItems().then(function(items) {
    for (var i in items) {
      if (items[i] instanceof XJS.GameItem) {
        // Manipulate your game item here
        items[i].setOfflineImage(path); // just an example here
      }
    }
  });
});
